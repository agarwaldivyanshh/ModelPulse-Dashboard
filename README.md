# ModelPulse Dashboard

## Getting started

Requirements: Node.js and npm — [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating) if needed.

```sh
npm i
npm run dev
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Scripts

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run preview` — preview production build
- `npm run lint` — run ESLint
- `npm run test` — run tests
